# Creator Portfolio
A React + Vite project for showcasing creative work.

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)

**Run it now in your browser:**  
👉 [creator-portfolio on GitHub Pages](https://talseg.github.io/creator-portfolio/)

---

### Prerequisites
- [git](https://git-scm.com/)
- [Node.js LTS](https://nodejs.org/)

### Installation
```bash
# Clone the repo
git clone https://github.com/talseg/creator-portfolio.git
cd creator-portfolio


# Install dependencies
npm install
# or
yarn install
```

### Run locally
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
